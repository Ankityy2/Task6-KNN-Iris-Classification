# Iris Flower Classification using K-Nearest Neighbors (KNN)

## Project Overview

This project demonstrates the implementation of the K-Nearest Neighbors (KNN) algorithm for classifying Iris flower species using the Iris dataset provided by Scikit-Learn.

The objective is to classify flowers into three species based on sepal and petal measurements.

## Dataset Information

Dataset Source:

* Scikit-Learn Built-in Iris Dataset

Dataset Size:

* Total Samples: 150
* Total Features: 4

Target Classes:

* Setosa
* Versicolor
* Virginica

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Google Colab

## Project Workflow

### 1. Dataset Loading

The Iris dataset was loaded directly from the Scikit-Learn library.

### 2. Data Exploration

* Checked dataset dimensions
* Converted dataset into a Pandas DataFrame
* Reviewed dataset statistics

### 3. Train-Test Split

The dataset was divided into:

* Training Set (80%)
* Testing Set (20%)

### 4. Model Training

A K-Nearest Neighbors (KNN) classifier was trained using the training data.

### 5. Prediction

The trained model predicted flower species on unseen test data.

### 6. Model Evaluation

Performance was evaluated using:

* Accuracy Score
* Classification Report
* Confusion Matrix

### 7. K Value Comparison

Different K values were tested to identify the most suitable number of neighbors for classification.

## Results

The KNN classifier achieved high classification accuracy and successfully identified flower species based on feature similarity.

## Key Concepts Learned

* K-Nearest Neighbors (KNN)
* Distance-Based Classification
* K Value Selection
* Accuracy Evaluation
* Classification Report
* Confusion Matrix

## Conclusion

A KNN classifier was successfully implemented using the Iris dataset. The model demonstrated excellent classification performance and provided practical understanding of distance-based machine learning algorithms.
